# JavaScript Assignment – Day-wise Aggregation

## Problem Statement
Given a dictionary where keys are dates (YYYY-MM-DD) and values are integers,
the task is to return a dictionary with days of the week as keys and the sum
of values corresponding to each day.

If a particular day is missing in the input, its value is calculated using
the sum of the previous and next available days.
